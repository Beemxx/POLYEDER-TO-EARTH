📐 Export ke CAD

Pilih format yang sesuai dengan software CAD Anda
📄
DXF — AutoCAD / BricsCAD
Format standar industri. Polygon dengan koordinat UTM presisi tinggi. Buka langsung di AutoCAD, BricsCAD, ZWCAD, LibreCAD.
⌨️
SCR Script — AutoCAD Command
Script perintah AutoCAD. Jalankan via Tools → Run Script. Cocok jika DXF tidak bisa dibuka langsung.
📊
CSV Koordinat — Excel / QGIS / Surpac
File CSV dengan kolom No, Label, Easting, Northing. Import ke Excel, QGIS, atau software survey lainnya.
